# Loopo 🟣
**Earn It. Flex It. Loop It.**

A gamified mission and rewards web app that teaches kids aged 8-14 financial 
responsibility through real-world tasks, earned credits, and a digital rewards 
marketplace.

## The Problem
Kids spend billions on Roblox and games but have no framework for earning, saving, 
or understanding value. Parents want tools that reinforce good behavior but existing 
apps are boring dashboards built for adults, not kids.

## The Solution
Loopo turns chores into missions. Kids earn credits by completing real tasks, parents 
approve and award credits, and kids redeem them for digital rewards they actually want 
(Roblox, Mobile Legends, Minecraft).

## Core Loop
Parent sets mission → Kid completes it → Parent approves → Credits awarded → Kid redeems rewards

## Built With
- React + Vite
- Tailwind CSS
- Supabase (auth, database, realtime)
- Lovable (AI-assisted development)
- Claude (planning, debugging, prompt engineering)

## AI Assistance
This project was built with significant AI assistance during the SEABW Hackathon 2025.

Lovable was used for AI-assisted code generation throughout development.

## How to Run
1. Clone this repository
2. Run `npm install`
3. Create a `.env` file with:
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key
4. Run `npm run dev`
5. Open http://localhost:5173
